# Bandit Level 0→1 - Reading File



## **Challenge:**



Password for the next level located in a readme file in the home directory



## **Solution:**



```bash
ls

cat readme
```

## **Explanation:**



The `ls` command lists files in the current directory, revealing a file called `readme`. Then the `cat` command prints the `readme` file to the terminal.
