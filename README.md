This repo documents my journey transitioning into DevOps, where I track my progress, consolidate my knowledge, and build practical, real-world skills.

## Purpose

The goal of this repository is to:

- Document what I learn in a structured way  
- Reinforce knowledge by solving real challenges  
- Build a deeper understanding through hands-on practice  
- Track my progress over time  
- Showcase my skills

Rather than just consuming content, I focus on:

- Breaking and fixing things  
- Understanding the “why” behind tools and concepts
